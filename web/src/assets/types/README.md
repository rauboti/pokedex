# Pokémon type icons

Colored type-badge SVGs vendored from
[partywhale/pokemon-type-icons](https://github.com/partywhale/pokemon-type-icons)
(MIT License — see `LICENSE` in this folder; © 2022 James Watkins).

One `<type>.svg` per canonical type (lowercase filename). Each is a self-contained
256×256 colored circular badge (colour baked in). Consumed by
`web/src/components/pokemon/TypeBadge.tsx`.
